An overview of the history of the web and the basics of web development

History of the World Wide Web

The Early Web

The World Wide Web was created by Tim Berners-Lee, with the original intent of sharing academic papers.

Documents on the internet, markup languages, and hypertext were brought together to form an early iteration of the web. Hypertext, inspired by Hypercard, was first used to link related academic papers on the web.

Further Contributions

Marc Andreesen created the first graphical web browser for common operating systems. Andreesen also proposed the img tag, allowing inline images to be displayed on the web for the first time.

Today, the W3C (World Wide Web Consortium) is responsible for recommendations and prototypes for web browsers and other web technologies.

HTML

What it is

HTML stands for Hypertext Markup Language. HTML is the set of markup symbols/codes in a file that is intended to be displayed on a web page. Each page element is enclosed in a tag/pair of tags. Tags are enclosed in angle brackets and usually come as a pair of opening and closing tags. Some tags (e.g. img, doctype, meta) are self-closing and do not have a separate closing tag. Most tags (not div or span) have semantic meaning and should be used to properly organize the webpage's content.

CSS

What it is

CSS stands for Cascading Style Sheets. CSS is a modified form of style sheets fit to control the visual presentation of a webpage. CSS is used to control the formatting, layout, and navigation of elements on a page, as well as the page as a whole. CSS files are linked to in the head of HTML files. An HTML file can have more than one CSS file.

CSS files are made up of rules defined for specific types of HTML elements. A rule has a selector, which designates the HTML elements affected, and a declaration, a declaration property and the declaration property's value. CSS rules follow the syntax: selector{declaration property: declaration value;}

Frameworks

What they are

Web frameworks are software platforms used for developing web application and sites. Web frameworks include pre-written components, code snippets, and application templates to streamline the development process.
